# 📊 Customer Churn Analysis Dashboard (Power BI)

## 📌 Project Overview

This project analyzes customer churn using the IBM Telco Customer Churn dataset. The dashboard provides interactive insights into customer behavior, churn trends, and key business drivers that impact customer retention.

---

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📈 Dashboard KPIs

- Total Customers
- Churn Customers
- Active Customers
- Churn Rate

---

## 📊 Dashboard Features

- Customer Churn by Contract Type
- Churn by Internet Service
- Churn by Payment Method
- Churn by Tech Support
- Churn by Online Security
- Churn by Gender
- Average Monthly Charges by Churn
- Churn by Tenure Group
- Customer Risk Segmentation
- Interactive Slicers

## 💡 Key Insights

- Overall churn rate is **26.54%**.
- Customers with **Month-to-Month contracts** have the highest churn.
- **Electronic Check** is the most common payment method among churned customers.
- Customers **without Tech Support** show significantly higher churn.
- Customers with **shorter tenure (<10 months)** are more likely to churn.
- Customers paying **higher monthly charges** have a greater tendency to churn.

---

## ✅ Business Recommendations

- Encourage customers to move to long-term contracts.
- Offer loyalty discounts to high-risk customers.
- Improve adoption of Tech Support services.
- Focus retention campaigns on new customers.
- Investigate reasons behind high churn among Fiber Optic users.
